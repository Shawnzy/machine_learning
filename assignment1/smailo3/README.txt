- Analysis Report is in smailo3-analysis.pdf
- Python code to analize cancer dataset is in cancer_dataset.ipynb
- Python code to analysis customer churn dataset is in churn_dataset.ipynb
- Customer churn raw dataset is in data_customer.csv
- List of all required libraries to run analysis code is in requirements.txt


To install all required libraries execute the following command in the terminal:
$ pip3 install -r requirements.txt

Use python 3.6 and  $jupyter notebook to run cancer_dataset.ipynb and churn_dataset.ipynb
